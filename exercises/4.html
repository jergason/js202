Exericse 4: POST

Use the materials/json_starter.html file you used previously.

Our server has an endpoint that accepts messages. If you send a POST to https://shielded-fjord-9233.herokuapp.com/messages/gdi-slc, a message will be added to the gdi-slc room.

The message should have these two fields:

var message = {
username: 'SOME USERNAME',
message: 'SOME MESSAGE'
}

Update your app to use $.ajax to POST your message to the server when the user clicks the button. You should still insert the message into the DOM as well.

Make sure to handle errors!
