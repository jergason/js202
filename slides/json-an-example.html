JSON: An example

{
 "firstName": "Jane",
 "lastName": "Smith",

   "streetAddress": "425 2nd Street",
   "city": "San Francisco",
   "state": "CA",

 },

Here is what JSON looks like! Keys are strings, values can be strings, other objects, arrays containing valid values, etc!
