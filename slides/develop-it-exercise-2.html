Develop It! Exercise 2

Use the materials/ajax.html file from the class exercises.

  • Create a function that uses XMLHttpRequest to get the list of rooms from the API and insert it into the DOM.
  • Attach a click handler to the button that calls your function
One theme you will notice over and over: PUT YOUR CODE IN FUNCTIONS. It should all be in functions. Functions make it easier to understand what your code is doing, easier to change, and easier to write.
